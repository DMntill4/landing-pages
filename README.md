## Descripción del Proyecto

El objetivo de este desarrollo es proporcionar un entorno de navegación intuitivo para una serie de landing pages experimentales y comerciales. La interfaz principal utiliza una paleta de colores en tonos azul profundo para garantizar una lectura cómoda y una estética moderna que prioriza el contenido de cada tarjeta.

## Arquitectura de Archivos

Para mantener la escalabilidad del proyecto, se ha seguido una organización de directorios estricta:

* **Raíz**: Contiene el acceso principal (`index.html`) y la configuración global de estilos (`style.css`).
* **Directorio SRC**: Aloja la lógica interna y los activos del sistema.
* **Directorio Landing**: Subdivisión donde cada proyecto reside en su propia carpeta independiente para evitar conflictos de dependencias y estilos.

---

## Estándares de Diseño

La construcción de estas interfaces se ha regido por los siguientes principios técnicos:

1. **Responsividad**: Adaptación fluida a diferentes resoluciones mediante el uso de unidades relativas (rem, em, %) y consultas de medios.
2. **Semántica**: Uso correcto de etiquetas HTML5 para mejorar la accesibilidad y el SEO técnico.
3. **Optimización**: Gestión eficiente de recursos externos y minimización de reglas redundantes en las hojas de estilo.
